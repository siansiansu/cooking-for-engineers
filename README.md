# Cooking for Engineers

工程師自煮指南 (持續更新)

0. 蔥、蒜、薑、辣椒切好裝在不同顏色的小盒子裡面，冰在冷藏或冷凍庫。
1. 不用煮飯的人負責洗碗。
2. 準備鮪魚、鯖魚、玉米、肉醬罐頭備用，注意不要太常吃。
3. 準備乾香菇、乾蝦米、菜脯、筍乾、肉鬆、樹豆、蓮子、鷹嘴豆等乾貨備用，注意不要太常吃。
4. 時間夠煮飯，時間不夠煮麵。
5. 冷凍肉一定要退冰。
6. 蔬菜水果用保鮮膜或是食材布包好放冷藏。
7. 滷肉、咖哩、湯可以前一晚下班後先煮好放冰箱冷藏，隔一個晚上也會比較好吃。
8. 蛤蠣用50度的溫水吐沙。
9. 不用太頻繁買菜，週末一次買好備齊。
10. IG 追蹤神人分享食譜，看到不錯的存成書籤看。
11. 裝菜跟裝肉的購物袋要分開。
12. 在傳統市場買菜買肉，不用講名字，直接跟老闆說你想煮什麼，老闆會幫你挑。
13. 砧板買三個，切菜一個、切肉一個、切水果一個。
14. 剪刀很好用。
15. 水果成熟的時間要錯開，冰箱深處放幾個蘋果備用。
16. 水餃很方便，但熱量很高。
17. 蛋便宜營養價值又高，農會有賣一箱30幾顆的。
18. 晚餐多煮一點明天午餐的份，裝在便當盒加熱一下就可以吃。
19. 小蘇打粉、過碳酸鈉選一個。
20. 料理用洗手乳選無味的慕斯。
21. 煮完抽油煙機不用馬上關。
22. 肉類泡鹹水去腥。
23. 抽油煙機的排油收集盒包保鮮膜，滿了丟掉。
24. 定時磨刀。
25. 掌握好鍋具熱鍋的時間。
26. 用計時器計時，不要用手機。
27. 肉類不要洗。
28. 排骨川燙去雜質。
29. 馬鈴薯買回來削皮切塊放冷藏或冷凍。
30. 蛋在桌子的平面處敲擊。
31. 收拾廚房包含洗碗、擦桌子、擦地板，不是洗完碗就沒事。
32. 用湯勺炒菜。
33. 擦試用品：衛生紙、廚房紙巾、酒精濕巾、一般濕巾。
34. 豆芽菜泡水放冷藏，要定時換水。
35. 香菇用撕的。
36. 青菜粗纖維用撕的。
37. 濾網是個好東西。
38. 在菜市場買肉請老闆幫忙切好、分裝小袋。
39. 放一瓶涼拌用橄欖油。
40. 蝦皮、蝦米、櫻花蝦爆香。
41. 爆香先薑後蒜。
42. 辣椒去籽可以去辣。
43. 先拔隱形眼鏡再切辣椒。
44. 煎荷包蛋鍋子太熱會冒泡。
45. 煎牛排步驟：煮前泡鹽水、四面煎熟鎖肉汁、旁邊放涼切塊、再下鍋煎或用烤箱至想要的熟度。
46. 菜瓜布至少三種：海綿、細纖維、粗纖維。
47. 兩人份平底鍋大小選擇：男生 -> 28 - 30 公分，女生 -> 24 - 26 公分
48. 不沾鍋是消耗品。
49. 鍋具買一體成型的。
50. 注意料理順序。
51. 蔥白蔥綠分開放。
52. 果醋調火鍋沾醬。
53. 夏天多做涼拌放冰箱。
54. 肉類順紋切、逆紋切口感不同。
55. 多吃不同種類的蔬菜水果肉類。
56. 玉米刀子下去，左右稍微滾動數次後，再用手折斷，不要硬切。
57. 切的時候注意沾板食材不要滑動，不然會受傷。
58. 滑嫩荷包蛋秘訣：鍋子溫度不要太高以及加一點水。
59. 貢丸不需要退冰。
60. 煮東西的時候不要關廚房的燈。
61. 同功能的鍋具鍋杓多準備幾隻，忙的時候沒時間洗。
62. 台灣在地農產品 > 國外進口食品。
63. 不高溫煎炒。
64. 使用不同品牌、不同原料的油品。
65. 刀不是越貴越好，勤保養保持鋒利小心使用比較重要。
66. 新手推薦使用肉桂刀。
